Here is a list of backend APIs you will need to create based on the .vue files in your project folder. These APIs will replace the hardcoded dummy data with dynamic data fetched from the backend. I have also included the request parameters and expected responses for each API.

1. Fetch All Teachings
Frontend Usage: Teachings.vue, Home.vue, SingleTeaching.vue
Endpoint: GET /api/teachings
Request Parameters:
search (optional): Search query string.
seriesId (optional): Filter by series ID.
tags (optional): Array of tags to filter by.
startDate (optional): Filter by start date.
endDate (optional): Filter by end date.
page (optional): Current page number.
limit (optional): Number of items per page.
Response:

{
  "teachings": [
    {
      "id": "1",
      "title": "Understanding God's Grace",
      "slug": "understanding-gods-grace",
      "excerpt": "A brief overview of God's grace.",
      "content": "<p>Full teaching content...</p>",
      "publishedAt": "2023-03-15T14:22:00Z",
      "readingTime": 10,
      "tags": ["grace", "faith"],
      "seriesName": "Foundations of Faith",
      "authorName": "Pastor Michael Johnson"
    }
  ],
  "pagination": {
    "page": 1,
    "totalPages": 5,
    "total": 50
  }
}

Fetch Single Teaching by Slug
Frontend Usage: SingleTeaching.vue
Endpoint: GET /api/teachings/:slug
Request Parameters:
slug (required): The slug of the teaching.
slug (required): The slug of the teaching.
{

5. Submit a Comment
Frontend Usage: CommentSection.vue
Endpoint: POST /api/comments
Request Body:

{
  "teachingId": "1",
  "name": "John Doe",
  "email": "john@example.com",
  "content": "This teaching was very insightful!"
}
Response:
{
  "id": "123",
  "teachingId": "1",
  "name": "John Doe",
  "email": "john@example.com",
  "content": "This teaching was very insightful!",
  "date": "2023-03-15T14:22:00Z"
}
6. Fetch Comments for a Teaching
Frontend Usage: CommentSection.vue
Endpoint: GET /api/comments
Request Parameters:
teachingId (required): The ID of the teaching.
Response:
{
  "comments": [
    {
      "id": "123",
      "name": "John Doe",
      "content": "This teaching was very insightful!",
      "date": "2023-03-15T14:22:00Z",
      "avatar": "https://example.com/avatar.jpg"
    }
  ]
}
. Newsletter Signup

Contact Form Submission
Frontend Usage: Contact.vue
